# Glossary of Terms

## A

### AI-robot Brain
The integrated system of AI perception, navigation, and control capabilities that enables intelligent robot behavior in humanoid robots.

### AprilTag
A visual fiducial marker system used for precise localization and navigation, optimized in Isaac ROS for GPU-accelerated detection.

## D

### Domain Randomization
A technique in simulation that randomizes environmental parameters to improve the transferability of AI models from simulation to reality.

## I

### Isaac ROS
A collection of packages and tools that enable hardware-accelerated perception for robots using NVIDIA GPUs, bridging traditional ROS 2 with modern AI frameworks.

### Isaac Sim
NVIDIA's robotics simulator built on Omniverse that provides photorealistic simulation environments for robotics development and synthetic data generation.

## N

### Nav2 (Navigation2)
The official navigation stack for ROS 2, providing path planning, execution, and obstacle avoidance capabilities specifically adapted for humanoid robots.

## S

### Synthetic Data
Artificially generated data that mimics real-world sensor data, created in simulation environments like Isaac Sim for AI model training.

## U

### USD (Universal Scene Description)
A 3D scene representation format developed by Pixar and used by NVIDIA Omniverse and Isaac Sim for scene description and collaboration.
